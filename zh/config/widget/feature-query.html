# 支持的查询类型

查询类型 查询类型描述 地图模式
Point 点查询 2D、3D
Circle 圆查询 2D
Rectangle 矩形查询 2D
Polygon 多边形查询 2D、3D
LineString 线查询 2D、3D
Cube 立方体查询 3D
PickModel 模型拾取查询(暂不支持) 3D
