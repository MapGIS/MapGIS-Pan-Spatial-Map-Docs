# 配置规范

// 这个文件描述添加数据微件配置信息
{
  // 托管IGServer服务器的ip
  "igsIp":"192.168.88.49",
  // 托管IGServer服务器的服务port
  "igsPort":"6163",
  // 托管IGServer服务器的文件上传port
  "igsUploadPort":"9999",
  "data":[
    {
      // 分类名称
      "name": "遥感影像",
      // 分类描述
      "description": "遥感影像",
      // 该分类下的数据列表
      "children": [
        {
          // 类型，IGSTile、IGSMapImage、IGSVector、IGSScene、OGCWMS、OGCWMTS、ArcGISMapImage、ArcGISTile、GEOJSON、KML、KMZ、CZML
          "type": "xxx",
          // url
          "url": "xxx",
          // 名称
          "name": "xxx",
          // 可选，令牌key，默认值为token，可自定义，比如天地图是tk
          "tokenKey": "xxx",
          // 可选，令牌value
          "token": "xxx"
        },
        // xxx
      ]
    },
    {
      // 分类名称
      "name": "地形图",
      // 分类描述
      "description": "地形图",
      // 该分类下的数据列表
      "children": []
    },
    //...
  ],
},

# 支持的数据类型

数据类型 数据类型描述 用到的参数 地图模式
OGCWMS OGC WMS 服务 type、url、name、tokenKey(可选)、token(可选) 2D、3D
OGCWMTS OGC WMTS 服务 type、url、name、tokenKey(可选)、token(可选) 2D、3D
ArcGISTile ArcGIS 瓦片服务 type、url、name、tokenKey(可选)、token(可选) 2D、3D
ArcGISMapImage ArcGIS REST 动态地图服务 type、url、name、tokenKey(可选)、token(可选) 2D、3D
IGSTile MapGIS 瓦片服务 type、url、name、tokenKey(可选)、token(可选) 2D、3D
IGSMapImage MapGIS 地图服务 type、url、name、tokenKey(可选)、token(可选) 2D、3D
IGSVector MapGIS 图层地图服务 type、url、name、tokenKey(可选)、token(可选) 2D、3D
IGSScene MapGIS 三维场景服务 type、url、name、tokenKey(可选)、token(可选) 3D
GEOJSON、KML、KMZ、CZML 其他 type、url、name 3D

# 数据占位文字

  • MapGIS REST XX 地图服务
地址示例：
动态地图服务：http://<server>:<port>/igs/rest/mrms/docs/{docName}
图层地图服务：http://<server>:<port>/igs/rest/mrms/layers?gdbps={gdbps}
瓦片地图服务：http://<server>:<port>/igs/rest/mrms/tile/{hdfName}
  • MapGIS 三维场景服务
地址示例：http://<server>:<port>/igs/rest/g3d/{modelName}
  • ArcGIS REST 动态地图服务
地址示例：http://<server>:<port>/arcgis/rest/services/ServiceRequest/MapServer
  • ArcGIS 瓦片服务
地址示例：http://<server>:<port>/arcgis/rest/services/ServiceRequest/MapServer
  • OGC WMS 服务
地址示例：http://<server>:<port>/igs/rest/ogc/doc/beijing/WMSServer
  • OGC WMTS 服务
地址示例：http://<server>:<port>/igs/rest/ogc/beijing/WMTSServer
