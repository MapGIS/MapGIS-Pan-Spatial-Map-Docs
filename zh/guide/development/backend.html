# 本地安装

  • 配置好数据库 MySQL5.7，创建数据库，默认数据库名为 mapgis_pan_spatial_map

    提示

    注意数据库编码为 utf8mb4,排序规则为 utf8mb4_general_ci

  • 在数据库上运行 apps/sql/mapgis_pan_spatial_map.sql

  • 修改配置文件

    复制 apps/config/application.example.properties 文件并命名为 application.properties，修改里面的服务端口和数据源信息

    # 服务端口
mapgis-pan-spatial-map.server.port=8015

# 数据源
mapgis-pan-spatial-map.datasource.db-host=localhost
mapgis-pan-spatial-map.datasource.db-port=3306
mapgis-pan-spatial-map.datasource.db-name=mapgis_pan_spatial_map
mapgis-pan-spatial-map.datasource.db-user=root
mapgis-pan-spatial-map.datasource.db-pwd=123456

    提示

    默认服务端口是 8015
    默认数据源端口是 3306，默认数据源 root 密码是 123456，请一定确保数据源信息与你配置的 MySQL 信息一致！！！

  • 双击执行startup.bat，浏览器访问http://localhost:8015/manager (opens new window)
