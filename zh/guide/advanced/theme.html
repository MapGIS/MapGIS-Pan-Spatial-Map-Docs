# 主题色

我们内置了一个色盘供您选择

如果这不能满足你的需求，你也可以使用任何你喜欢的颜色，只需要在 packages/pan-spatial-map-framework/src/config/config.js 文件中配置你的主题色即可。如：



 




module.exports = {
  theme: {
    color: '#13c2c2', //换成任何你喜欢的颜色，支持 hex 色值
    mode: 'night'
  }
}

当你设置好主题色后，系统会根据这个主题色为你生成一系列配套的颜色，并应用到 vue 组件中。

提示

你可以在你的样式文件中直接使用 less 变量 @primary-color。

注意

主题色目前只支持 hex 模式的色值。如果设置为 rgb 或其它模式的色值，可能会导致配套颜色无法生成。

# 功能色

除了主题色，系统还有一些功能性颜色，分别为：成功色、警告色和错误色。默认色值分别为：

名称 success warning error
色值 #52c41a #faad14 #f5222d
颜色
less 变量 @success-color @warning-color @error-color

你也可以在 src/config/config.js 重新定义这些功能色





 
 
 



module.exports = {
  theme: {
    color: '#13c2c2',
    mode: 'night',
    success: '#52c41a', //定义成功色，支持 hex 色值
    warning: '#faad14', //定义警告色，支持 hex 色值
    error: '#f5222d' //定义错误色，支持 hex 色值
  }
}

提示

想在在你的样式文件中使用以上各功能色，引用各功能色对应的 less 变量即可。

注意

功能色目前也只支持 hex 模式的色值。如果设置为 rgb 或其它模式的色值，可能会导致配套颜色无法生成。

dark / 暗色菜单模式:

你可以在这三种模式之间随意切换，也可以在 src/config/config.js 中设置默认的主题模式。




 



module.exports = {

# 动态切换

在地图页面packages/pan-spatial-map-framework/src/pages/map/Map.vue中拿到 app.json (opens new window) 中的 themeStyle 后动态地设置到应用中。























 



























export default {
  data() {
    return {
      application: {}
    }
  },
  computed: {
    ...mapState('setting', ['theme'])
  },
  async created() {
    const appInfo = await getAppInfo()

    await AppManager.getInstance().loadConfig(
      BASE_URL,
      appInfo.data.configPath,
      appInfo.data.assetsPath
    )

    this.application = AppManager.getInstance().getApplication()

    const style = this.themeStyle()

    this.setTheme({ ...this.theme, mode: style.theme, color: style.color })
  },
  methods: {
    ...mapMutations('setting', ['setTheme']),
    themeStyle() {
      if (this.application.theme) {
        if (this.application.theme.style) {
          if (this.application.theme.manifest) {
            const style = this.application.theme.manifest.styles.find(item => {
              return item.name === this.application.theme.style
            })

            if (style) {
              return {
                color: style.color,
                theme: style.theme
              }
            }
          }
        } else if (this.application.theme.customStyle) {
          return this.application.theme.customStyle
        }
      }
      return { theme: 'dark', color: '#1890ff' }
    }
  }
}

# 色弱模式

对于有视觉障碍的群体，我们提供了色弱模式，你可以通过配置 src/config/config.js 启用色弱模式






 


module.exports = {
  theme: {
    color: '#13c2c2',
    mode: 'night'
  },
  weekMode: false //色弱模式，true:开启，false:不开启
}

完整的系统设置参考 src/config/default/setting.config.js

提示

以上所有主题设置项，均已映射到 vuex/setting 模块的 state 中，你可以通过提交 setting/mutations 实时修改设置项。

# 更换软件主图标

  1. 图片放置位置
    将.ico 图片放到 packages/pan-spatial-map-framework/public 文件夹下；
    将.png 文件放到 packages/pan-spatial-map-framework/src/assets/img 文件夹下。

  2. 更换浏览器标签图标
    修改 packages/pan-spatial-map-framework/public/index.html

<link rel="icon" href="<%= BASE_URL %>logo-earth.ico" />
  1. 更换登录页面图标
    修改 packages/pan-spatial-map-framework/src/pages/login/Login.vue
<img alt="logo" class="logo" src="@/assets/img/logo-earth.png" />
  1. 更换软件标题图标
    修改 packages/pan-spatial-map-mock-server/mock/data/app.json; 这个可以直接通过后台上传图片修改。
  "logo": "<svg xxx"/></svg>"
  1. 更换关于界面图标
    修改 packages/pan-spatial-map-framework/src/components/About/About.vue
<img alt="logo" class="logo" src="@/assets/img/logo-earth-blue.png" />

# UI

  1. 修改主题背景透明度
    修改 mapgis-pan-spatial-map-pro-antd\packages\pan-spatial-map-framework\src\config\default\antd.config.js 中，各主题对应的body-background 和component-background 属性，给这个两个属性加上透明度即可。还可以根据需要修改其他属性。

  1. 修改窗口大小
    左边侧边区：设置 MpPanSpatialMapSideWindow 的 width 值。
    可参考全空间一张图里在 packages/pan-spatial-map-plugin-theme/src/themes/classic-theme/classic-theme.vue 对 width 的设置。

  2. 修改工具面板大小
    在 app.json 里设置 panel 的 styles。不设置默认自适应内容大小。

{
  "panel": {
    "position": {
      "anchor": "top-right",
      "horizontalOffset": 10,
      "verticalOffset": 52
    },
    "styles": {
      "width": 400,
      "height": 480
    }
  }
}
  1. 修改结果集面板大小
    设置 packages/pan-spatial-map-plugin-theme/src/components/Footer/Footer.vue 的 maxViewHeight 值，默认 400。
    可参考全空间一张图里在 packages/pan-spatial-map-plugin-theme/src/themes/classic-theme/classic-theme.vue 对 maxViewHeight 的设置。
